# 🚀 Zorvyn Dashboard | Financial Management System

A high-performance, visually stunning financial dashboard built for speed and security. **Zorvyn** combines a sleek, modern UI with real-time data synchronization and enterprise-grade animations.

## ✨ Key Features

*   **Secure Authentication:** Full login/signup flow powered by **Firebase Auth**.
*   **Smooth Motion UI:** Enterprise-grade interactions and page transitions powered by **GSAP**.
*   **Responsive Design:** Fully optimized for Desktop, Tablet, and Mobile views.
*   **Interactive Widgets:** Dynamic charts and transaction logs for clear financial insights.

## 🛠️ Technical Stack


| Category | Technology |
| :--- | :--- |
| **Frontend** | HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+) |
| **Animation** | GSAP (GreenSock Animation Platform) |
| **Backend** | Firebase (Authentication & Firestore) |
| **Icons** | Lucide Icons / FontAwesome |
| **Hosting** | Firebase Hosting / Vercel |
